Welcome to NIT-Warangal

This is the website for opensource wing of NIT-Warangal. You can find our awesome opensource projects here. We shall soon start writing about our projects and how you can start contributing to us.

You need to have git bash installed on your system. For ease of use, we recommend Operating Systems like Ubuntu or any debian based linux distro.
